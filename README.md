# Sorting-and-Searching-Visualizer

An interactive web-based Algorithm Visualizer built using HTML, CSS, and JavaScript to help learners understand sorting and searching algorithms. The application supports Bubble, Insertion, Selection, Merge, and Quick Sort, along with Linear and Binary Search, featuring step-by-step execution, adjustable animation speed, recursion visualization, and real-time complexity insights. Designed with a responsive UI and clear visual cues, this project focuses on making core DSA concepts intuitive and engaging for beginners.
